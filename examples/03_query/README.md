# Query API Example

This example demonstrates the usage of the `Query` API using a simple Math grammar.

It shows how to:
1.  Define a grammar (`gen.nim`) and generate a parser.
2.  Parse input into a syntax tree.
3.  Create a `Query` object with S-expression patterns.
4.  Execute the query to find matches and captures.

## Run

```bash
nim r gen.nim
nim r main.nim
```
